# SSH Switch Configuration Lab

## Objective
Secure remote access to a network switch using SSH.

## Skills Demonstrated

- SSH configuration on Cisco switches
- Basic switch security configuration
- CLI configuration and verification

## Tools Used

Cisco Packet Tracer

## Key Concepts

Secure Shell (SSH) provides encrypted remote access to network devices. Configuring SSH improves network security compared to unsecured protocols such as Telnet.

## Validation

Successful SSH access to the switch was verified using:

- remote login tests
- CLI authentication
- connection verification
