# Router Interface Configuration Lab

## Objective
Configure router interfaces and verify connectivity between connected networks.

## Skills Demonstrated

- Router interface configuration
- IPv4 addressing
- Network verification
- CLI configuration using Cisco IOS

## Tools Used

Cisco Packet Tracer

## Key Concepts

Routers connect multiple networks together and route traffic between them. Correct interface configuration and IP addressing are essential for ensuring network connectivity.

## Validation

Connectivity between networks was verified using:

- ping
- show ip interface brief
- interface status checks
